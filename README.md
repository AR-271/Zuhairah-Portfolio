# Nur Zuhairah – Portfolio

Pink/rose-themed personal portfolio. Responsive for both desktop and mobile.

## Folder Structure

```
nurzuhairah-portfolio/
├── index.html
├── README.md
└── images/
    ├── hero-bg.png            ← Home background (desktop/landscape)
    ├── hero-bg-mobile.png     ← Home background (mobile/portrait)
    ├── about-bg.png           ← About background (desktop)
    ├── about-bg-mobile.png    ← About background (mobile)
    ├── exp-bg.png             ← Experience background (desktop)
    ├── exp-bg-mobile.png      ← Experience background (mobile)
    ├── contact-bg.png         ← Contact background (desktop)
    ├── contact-bg-mobile.png  ← Contact background (mobile)
    └── exp/                   ← Experience card photos (add when ready)
        ├── exp-01.jpg         ← Kiosk/Stall Salesperson – Iradah Hijab
        ├── exp-02.jpg         ← Intern – Balai Khazanah Islam
        ├── exp-03.jpg         ← Caretaker – PGGMB Daycare
        ├── exp-04.jpg         ← Intern – Pusat Sejarah Brunei
        └── exp-05.jpg         ← Caretaker – Yadira Daycare Centre
```

## Adding Experience Card Photos

1. Rename the photo to the matching filename (e.g. `exp-01.jpg`)
2. Place it inside `images/exp/`
3. The card automatically shows your photo — no code changes needed
4. If a photo is missing, the animated sky + clouds shows instead

## Deploying to GitHub Pages

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages → Source**: `main` branch, `/ (root)`
3. Live at `https://yourusername.github.io/repo-name`
